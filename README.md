# Webpack for react
> 适合`react`开发的`webpack`基础项目

## 功能

- [x] http服务
- [x] 热加载
- [x] css、图片等常用文件处理
- [x] http代理
- [x] sass与less文件处理
- [x] babel编译es6
- [x] `.js`、`.css`、`.scss`、`.less`文件调试
- [x] `.jsx`文件处理
- [x] `.jsx`文件调试
- [x] 打包应用于生产环境的静态文件
- [x] css作用域组件化

## 安装与运行

```bash
# 安装依赖
$ npm i

# 运行
$ npm run start

# 调试模式运行
$ npm run dev

# 打包静态文件
$ npm run build
```
